- **Thao tác trong-app** (đánh dấu In một tài liệu → phát `khoChanged`): Home dựng lại cây (`buildSnapshot #2`) → "Đi in 1" hiện → gỡ cờ → block biến mất. Cache invalidate đúng.

- `src/storage/khoSnapshot.ts`: một walk toàn kho → cây RAM (KHÔNG ghi file cache vào Syncthing); `listMon`/`summarizeMon`/`countPrintFlagged`/`listInboxByMon`/`listReading` dẫn xuất từ cây (nội dung `_reading-*.json` vẫn đọc tươi mỗi lần cho tiến độ).
