## Task 5: Cài đặt — Cỡ chữ Viewer (base-scale, không đụng pinch)

**Files:** `src/viewer/fontScale.ts` (mới), `src/pages/SettingsPage.tsx`, `src/pages/ViewerPage.tsx`, `src/components/PdfView.tsx`

- [ ] `src/viewer/fontScale.ts`:
```ts
import { Preferences } from '@capacitor/preferences';
const KEY = 'viewer_base_scale';
export const SCALE_OPTIONS = [
  { label: 'Vừa', value: 1 },
  { label: 'Lớn', value: 1.5 },
  { label: 'Rất lớn', value: 2 },
];
export async function getBaseScale(): Promise<number> {
  const { value } = await Preferences.get({ key: KEY });
  const n = value ? parseFloat(value) : 1;
  return Number.isFinite(n) && n >= 1 ? n : 1;
}
export async function setBaseScale(v: number): Promise<void> {
  await Preferences.set({ key: KEY, value: String(v) });
}
```
- [ ] SettingsPage: mục "Cỡ chữ Viewer" dùng `IonSegment` (value = String(scale)), load `getBaseScale()`, onIonChange → `setBaseScale(parseFloat(value))`. Labels Title case từ SCALE_OPTIONS.
- [ ] ViewerPage: trong effect load, đọc `const base = await getBaseScale();` (cạnh getResumePage), truyền `baseScale={base}` cho PdfView (chỉ render PdfView khi `ready` — đã có pattern; thêm base vào điều kiện ready hoặc default 1).
- [ ] PdfView: thêm prop `baseScale?: number` (default 1). Khởi tạo `useState(baseScale)` cho zoom; `zoomRef`/`prevZoom` init `baseScale`. **pinch min vẫn 1** (cho thu về fit-width); chỉ initial = baseScale. Mọi chỗ khác giữ nguyên (windowing/offsets/anchor dùng `zoom` hiện tại → tự đúng).
- [ ] **RÀNG BUỘC:** mở tài liệu render đúng cỡ base; pinch phóng/thu mượt trên nền đó; nhớ-trang đúng (device verify Task 7).
