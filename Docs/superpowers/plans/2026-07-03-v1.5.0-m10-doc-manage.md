# v1.5.0 — M10 phần 1: quản lý tài liệu cấp file (đổi tên / chuyển / xóa + tầng vuốt) — Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans. Steps dùng checkbox.

**Goal:** Vuốt trái hàng tài liệu (FolderPage) → In · Xóa · ⋯; ⋯ mở sheet 4 hành động (Đổi tên hiển thị / Chuyển tới… / In / Xóa). Đổi tên = companion `.display.json` (override, không rename file thật). Chuyển = dời trọn cụm file + reading-state máy mình. Xóa = xóa trọn cụm + reading máy mình. KHÔNG đụng worker; folder rows + pending không có action.

**Architecture:**
- **Tên hiển thị:** companion `<base>.display.json` `{name}` cạnh cặp, khuôn `.print.json`. Ưu tiên: `.display.json` > tên file (CHỐT: không đọc sidecar title). `classify` bỏ qua `.display.json` + đính `displayUri` cho Document; nơi liệt kê đọc companion (chỉ doc có → ít read) để ra tên. Sync qua Syncthing như file kho.
- **Cụm tài liệu:** mọi file trong folder có `name === base` hoặc `name.startsWith(base + '.')` → `<base>.pdf` + `<base>.json` + `<base>.print.json?` + `<base>.display.json?`.
- **Chuyển = copy+delete** bằng primitive sẵn (KHÔNG native move mới): PDF `copyToDir` (stream, mime pdf — an toàn), các file json `readFile`+`writeFile` (text — tránh bug `.tmp` octet-mime Samsung), rồi `deleteFile` bản gốc. **Dedup hợp nhất:** tính `newBase` không đụng ở đích cho CẢ cụm (đích trùng → `(k)` đồng bộ trước đuôi), ghi tên đã-unique nên không auto-dedup lệch. Reading-state máy mình: `moveReading(oldPath→newPath)`.
- **Xóa = `deleteFile` từng file cụm + `removeReading(oldPath)`** (companion `.print.json`/`.display.json` xóa theo → Đi in/tên tự cập nhật).
- **Đích Chuyển = tái dùng `ChooseMonSheet` v1.3.0**, mở rộng `onPick(path, destUri)` (destUri = folder-uri đích; import bỏ qua destUri) — không fork.
- Action chỉ trên `documents` (cặp pdf+json đã xử lý). Pending (⏳ lẻ) + folder: KHÔNG action → tránh ca cụm lệch.

**Tech Stack:** Ionic React (IonItemSliding vuốt, sheet thẻ-rời, IonAlert confirm), Saf primitives sẵn, reading model. Vitest cho logic thuần (display parse, cluster, dedup).

**Bối cảnh đã xác minh:**
- `classify.ts`: `Document{name,pdfUri,jsonUri,printFlagged}`; skip `_`/`.`-prefix + `.print.json`.
- `repo.listFolder(uri)→FolderListing{folders,documents,pending}`; `createSubfolder(parentUri,name)` (`Saf.createDir` reject 'exists', **return void → sẽ đổi return uri**).
- `Saf`: copyToDir(stream+dedup(k)-trước-đuôi+mime-đuôi), readFile(text), writeFile(dir,name,content), deleteFile(uri), listFolder.
- reading `store.ts`: `recordProgress`, `removeReading(path)`, `getResumePage`, `listReading` (union + `resolveUriFromRelPath`); `model.ts`: `mergeReading/upsertEntry/removeEntry`; `paths.ts`: `relPathFromUris(root,uri)`.
- FolderPage document row hiện: `<IonItem button onClick=viewer>` + `<PrintFlagButton>` luôn-hiện slot=end.
- ChooseMonSheet `onPick(path:string[])`; drill có `stack:{name,uri,folders}[]`; UNFILED card `onPick([UNFILED])` (chưa có uri); sidecar CÓ `title` nhưng CHỐT không dùng.

```ts
export const DISPLAY_SUFFIX = '.display.json';

export function parseDisplayName(json: string): string | null {
  try {

    return n || null;
  } catch { return null; }
}
```

```ts
// Đuôi các file cùng cụm của tài liệu <base> trong danh sách tên (name===base+suffix).
export function clusterSuffixes(base: string, names: string[]): string[] {
  const out: string[] = [];
  for (const n of names) if (n.startsWith(base + '.')) out.push(n.slice(base.length));
  return out;
}
// base không đụng ở đích cho CẢ cụm: né mọi <cand><suffix> trong existing. dedup "(k)" trước đuôi.
export function uniqueBase(base: string, existing: string[], suffixes: string[]): string {
  const has = (cand: string) => suffixes.some((s) => existing.includes(cand + s));
  if (!has(base)) return base;
  for (let k = 1; k < 1000; k++) { const cand = `${base} (${k})`; if (!has(cand)) return cand; }
  return base;
}
```

```ts
import { Saf } from '../plugins/saf';
import { getRootUri } from './repo';
import { emitKhoChanged } from '../lib/khoEvents';
import { clusterSuffixes, uniqueBase } from './docCluster';

const isBinary = (suffix: string) => suffix.toLowerCase() === '.pdf';

// Các entry (name,uri) cùng cụm của tài liệu base trong folderUri.
async function clusterEntries(folderUri: string, base: string): Promise<{ name: string; uri: string; suffix: string }[]> {
  const { entries } = await Saf.listFolder({ uri: folderUri });
  const names = entries.filter((e) => !e.isDirectory).map((e) => e.name);
  const suffixes = clusterSuffixes(base, names);
  return entries
    .filter((e) => !e.isDirectory && suffixes.includes(e.name.slice(base.length)))
    .map((e) => ({ name: e.name, uri: e.uri, suffix: e.name.slice(base.length) }));
}

// Đặt tên hiển thị (override). name rỗng → xoá companion (về tên file).
export async function setDisplayName(folderUri: string, base: string, name: string): Promise<void> {
  const trimmed = name.trim();
  const { entries } = await Saf.listFolder({ uri: folderUri });
  const existing = entries.find((e) => !e.isDirectory && e.name === `${base}.display.json`);
  if (trimmed) {
    await Saf.writeFile({ dirUri: folderUri, name: `${base}.display.json`, content: JSON.stringify({ name: trimmed }) });
  } else if (existing) {
    await Saf.deleteFile({ uri: existing.uri });
  }
  emitKhoChanged();
}

// Chuyển trọn cụm sang folder đích. Trả base mới ở đích (để caller dời reading-state).
export async function moveDocument(folderUri: string, base: string, destFolderUri: string): Promise<string> {
  const cluster = await clusterEntries(folderUri, base);
  const { entries: destEntries } = await Saf.listFolder({ uri: destFolderUri });
  const destNames = destEntries.filter((e) => !e.isDirectory).map((e) => e.name);
  const suffixes = cluster.map((c) => c.suffix);
  const newBase = uniqueBase(base, destNames, suffixes);
  for (const f of cluster) {
    const newName = newBase + f.suffix;
    if (isBinary(f.suffix)) {
      await Saf.copyToDir({ srcUri: f.uri, dirUri: destFolderUri, name: newName }); // stream, mime pdf, tên đã unique → không auto-dedup
    } else {
      const { data } = await Saf.readFile({ uri: f.uri });
      await Saf.writeFile({ dirUri: destFolderUri, name: newName, content: data });
    }
  }
  for (const f of cluster) await Saf.deleteFile({ uri: f.uri }); // xoá gốc sau khi copy xong cả cụm
  emitKhoChanged();
  return newBase;
}

export async function deleteDocument(folderUri: string, base: string): Promise<void> {
  const cluster = await clusterEntries(folderUri, base);
  for (const f of cluster) await Saf.deleteFile({ uri: f.uri });
  emitKhoChanged();
}

// relPath thư mục (dùng dựng path reading mới). null nếu không suy được.
export { getRootUri };
```
