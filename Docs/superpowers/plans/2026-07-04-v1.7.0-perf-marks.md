# v1.7.0 — Perf marks + màn debug (đo, KHÔNG sửa)

**Goal:** Gắn thước đo rẻ (in-memory) cho 6 luồng xương sống + màn debug trong Cài đặt để lập baseline trên máy thật. Không đổi hành vi luồng chính.

**Architecture:** Một module `src/perf/perf.ts` giữ số đo trong RAM (Map start-times + mảng mẫu/luồng, cap 30). Mỗi luồng có một điểm `start` và một điểm `end` neo ở **mốc người dùng cảm nhận (vẽ xong)**. Marks = `performance.now()` diff → không profiling nặng, không observer effect. Màn debug = IonModal mở từ Settings: bảng last/min/max/median + nút copy text + reset.

**Tech:** React/Ionic, TypeScript. TDD cho phần thuần (median/record/stats/report).

## Anchor map (điều tra)
| Luồng | start | end (mốc vẽ xong) |
|---|---|---|
| coldStart | (nav origin = 0) | HomePage lần paint đầu → `perfColdReady()` = `performance.now()` |
| openMon | FolderPage mount effect | `listing` paint đầu (double-rAF) |
| openDoc | ViewerPage mount | PdfView `onRenderSuccess` đầu |
| zoomCommit | `beginSwap()` | `pendingRender` rỗng (nét); huỷ start khi timeout 1.5s |
| importBatch | `onPick` (trước copy) | sau `importBatch()` resolve (⏳ vào _inbox) |
| enterSelect | long-press fire (sau 450ms hold) | `selectMode` render (double-rAF) |

Ghi chú neo: 3 luồng "mở" đo work app-controllable (đọc/vẽ), KHÔNG gồm hoạt ảnh chuyển trang (Ionic cố định) và (coldStart) khởi động native trước WebView — ghi rõ trong màn debug.
